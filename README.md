# Youtube-Content-Monetization-Modeler
The Content Monetization Modeler is a machine learning project designed to predict YouTube ad revenue based on video performance metrics and contextual factors.

The Content Monetization Modeler is a machine learning project designed to predict YouTube ad revenue based on video performance metrics and contextual factors.

Dataset Name: YouTube Monetization Modeler

Format: CSV

Size: ~122,000 rows

1. Data Exploration(EDA)
   -Analyzed data distribution
   -Identified correlations
   -Detected Outliers
   -Visualized enagement metrics
2. Data cleaning
   - Handling mssing values
   - Removed duplicates
3. Feature Engineering
   - Enagement rate
   - Like rate
   - Watch time per user
   - Subscriber enagement
4. Categorical encoding
   - Category
   - Device
   - Country
5. Feature scaling
   - Applied Standard scaler

Five Machine learning models
1. Linear regression
2. Polynomial regression
3. Decision Tree regression
4. Random forest regression
5. K-Nearest Neighbors

Models evaluation metrics
1. R² Score
2. Root Mean Squared Error (RMSE)
3. Mean Absolute Error (MAE)
